# agents_md

A collection of tools, skills, and memo documents for AI agents.

## Overview

This repository collects useful resources, techniques, and reference notes that help AI agents perform better at work. It is designed for developers, architects, and agent builders who want a practical knowledge base for building more capable and productive AI systems.

## Contents

- Agent skill templates
- Prompt patterns and best practices
- Integration notes for tools and APIs
- Workflow cheat-sheets
- Troubleshooting and optimization tips

## Purpose

The goal is to make AI agents "insane at work" by providing a centralized set of resources for:

- task automation
- context management
- decision making
- tool orchestration
- memory and state handling

## How to use

1. Browse the documents for the skill or workflow you need.
2. Copy and adapt prompts, code snippets, and templates.
3. Combine resources to create better agent behavior.

## Contribution

Add new skills, improve existing notes, and keep the collection up to date with the latest agent patterns.
